# centos7-gcc7.x
Docker image based on `CentOS7`, installed `gcc-7.4.0` compiler and running libraries(`libstdc++`, etc.).    

## Build Image
```bash
cd docker/centos7-gcc7.x
docker build -t wangyoucao577/centos7-gcc7.4 .
```

## Installed Tools
- `gcc-7.4.0`
    - `libstdc++.so.6.0.24`   
- `git-2.x`    
- `cmake-3.14.1`
